
---

## Prerequisites

- **Node.js** (v16+ recommended)
- **npm** (comes with Node.js)

---

## Setup Instructions

1. Clone the repository or copy the project folder locally:

```
git clone <repo-url>
cd project
```

2. Install Dependencies
```
npm install express body-parser cors
```

3. Start the Server
```
node server.js
```

4. Visit this url in your browser
```
http://localhost:3000
```
